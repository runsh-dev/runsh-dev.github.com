01

本季完成

主色从浅葱（#33A6B8）替换为梅红（#C56473），与暖灰中性体系同族；中性灰从冷灰迁移到 60% 暖度的纸张色，30 项 token 已稳定。

Components 与 Snippets 两节完成与 apps/web 实际样式的对齐：按钮采用 accent/8 半透 + 30% 边框，输入框用 --field-* 全套，评论气泡用异形圆角。

02

色彩与字体

主色：梅红 #C56473（深色模式 #E095A4）。语义色采用日本传统色：縹（信息）、若竹（成功）、朽葉（警告）、蘇芳（错误）。

中性灰 n-1 → n-10（暖色调，60% 暖度）

字体三族：sans / serif / mono，每族都自带完整中文回退链。中文不施合成 bold，font-medium（500）即上限。

Sans：Inter → PingFang SC → Noto Sans SC。Serif：Noto Serif CJK SC → SongTi SC → Georgia。

减法不是少做事，是每一次加法都被审过。
